Add another sprite later

1. Drop the PNG into sprite-widget/sprites/

2. Add a new entry to the sprites array below — that's it.

SpriteAnimator.mount("#widget", {
  sprites: [
    { id: "beholder", label: "Beholder",
      src: "sprites/BEHOLDER.png",
      frames: 8, frameWidth: 256, frameHeight: 256, fps: 8 },

    // future additions:
    // { id: "wraith", label: "Wraith",
    //   src: "sprites/WRAITH.png",
    //   frames: 6, frameWidth: 128, frameHeight: 128, fps: 10 },
  ],
  defaultId: "beholder"
});

The dropdown at the top of the widget appears automatically and lets visitors switch sprites. You can also call widget.addSprite(...) at runtime.
